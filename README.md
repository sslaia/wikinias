# WikiNias

An Android app for Nias Wikipedia, Wiktionary and Wikibooks. Get it from Play Store:

<a href="https://play.google.com/store/apps/details?id=com.blogspot.wikinias"><img alt="Get it on Google Play" src="https://play.google.com/intl/en_us/badges/images/apps/en-play-badge.png" height="80pt"/></a>

## Version's history

### Version 2.5.0:
- Massive code spring cleaning
- Made the app core modular (package extraction)
- Made the language and project selection persist
- Adapted to screen orientation and various screen sizes
- Improve image loading and search results page
- Pronunciation audio works
- New article bookmarks feature

### Version 1.0.0:

- First release
- Support for Nias Wikipedia, Wiktionary and Wikibooks
- Dark/light theme
- Font size settings

## Getting the app

WikiNias app is live on [Google Play Store](https://play.google.com/store/apps/details?id=com.blogspot.wikinias).
