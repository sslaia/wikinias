# WikiNias

**WikiNias** is a Flutter application dedicated to the preservation and learning of the Nias language and culture. This repository contains the source code for the app and the static landing page hosted via GitHub Pages.

## ðŸŒ Setting up the Landing Page

To make the `index.html` file visible as a website:

1.  Go to the **Settings** tab of this repository.
2.  Click on **Pages** in the left sidebar.
3.  Under **Build and deployment** > **Branch**, select `main` (or `master`) and ensure the folder is set to `/ (root)`.
4.  Click **Save**.
5.  Your site will be live at: `https://sslaia.github.io/wikinias/`
